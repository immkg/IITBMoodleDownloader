# IITBMoodleDownloader
Download all files from the IIT Bombay Moodle news forum in a single command

Works for the new moodle website only!

REQUIREMENTS:

1. Python 2.7+

2. Twill: 

        sudo apt-get update

        sudo apt-get install python-twill

3. tqdm: 

        pip install tqdm

4.requests:
        
        pip install requests


COMMAND:

        python download.py
